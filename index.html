📈 Modelado de Incidentes de TI

Estructura de tablas SQL para la minería de datos de la Mesa de Ayuda, permitiendo identificar patrones de fallas repetitivas en hardware.

SELECT aula, COUNT(id_ticket) AS Fallas_Reportadas
FROM Tickets_TI
WHERE estado = 'Cerrado'
GROUP BY aula
ORDER BY Fallas_Reportadas DESC;
